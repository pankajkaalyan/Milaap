2. You'll see the search filters page
3. Page title says "Search Profiles" or "Find Matches"

#### Step 2: Apply Filters

A filter is a criterion that narrows down results. You'll see many filter options:

**Filter 1: Age Range**
- Find people between age X and Y
- Click on age fields
- Enter minimum and maximum age
- Example: 24 to 32 years

**Filter 2: Height Range**
- In centimeters or feet
- Example: 160 cm to 180 cm (or 5'4" to 5'11")

**Filter 3: Location**
- Select preferred cities/states
- Multi-select (can choose multiple)
- Example: Mumbai, Pune, Delhi

**Filter 4: Religion**
- Select which religions acceptable
- Multi-select
- Example: Hindu, Sikh, Jain

**Filter 5: Caste**
- Select specific castes
- Example: Brahmin, Sharma, Patel

**Filter 6: Education**
- Minimum education required
- Example: Graduate (Bachelor's degree)

**Filter 7: Profession**
- Select acceptable professions
- Example: Engineer, Doctor, Management

**Filter 8: Marital Status**
- Never married, Divorced, Widowed, Separated
- Example: Only never married

**Filter 9: Complexion**
- Fair, Wheatish, Dark
- Example: Any

**Filter 10: Body Type**
- Slim, Athletic, Average, Heavy
- Example: Slim to Athletic

**Filter 11: Income (Optional)**
- Desired income range
- Example: 10-15 LPA
- Can be left blank

**Filter 12: Interests**
- Common interests
- Example: Travel, Reading, Sports

#### Step 3: Set Your Filters

1. For each filter, decide what you want
2. Check boxes or enter ranges
3. You can skip filters you don't care about
4. Leave blank = no filter applied

#### Step 7: Modify Search

**Want different results?**
1. Scroll back to top
2. Find filters section
3. Change one or more filters
4. Click "Search" again

**Examples of modifications:**
- Expand age range from 25-30 to 23-32

### Understanding Search Results

#### Search Results Count

- **0 results**: No one matches your criteria
  - Solution: Broaden your filters
  - Expand age range
  - Add more cities
  - Add more religions/castes

- **1-5 results**: Very specific criteria
  - Solution: Maybe widen filters slightly
  - Or keep criteria and check results regularly (new people join daily)

- **5-20 results**: Good number
  - Manageable to browse
  - People match your preferences well

- **20-50 results**: Larger pool
  - More options
  - May need to narrow down

- **50+ results**: Very broad criteria
  - Can see top matches first (sorted by compatibility)
  - Can narrow filters to reduce

#### Match Percentage

What it shows:
- How well profile matches YOUR preferences
- Based on age, location, religion, caste, etc.
- NOT based on how likely they'll like you back

**Interpretation:**
- 95-100%: Perfect match for your criteria
- 80-94%: Excellent match
- 70-79%: Good match
- 60-69%: Acceptable match
- Below 60%: Doesn't match preferences well (but might still be worth checking!)

**Important Note:**
- High match % doesn't guarantee chemistry
- Low match % doesn't mean incompatibility
- Check profiles regardless of %
- Sometimes best connections are unexpected!

### Search Tips and Tricks

1. **Start Broad**
   - Don't apply too many filters initially
   - See what's available
   - Then refine

2. **Update Regularly**
   - New users join every day
   - Re-run searches weekly
   - You might find new people

3. **Check Recently Updated**
   - Recently updated profiles rank higher
   - More likely to respond
   - More active users

4. **Use Multiple Searches**
   - Save multiple search criteria
   - "Young professionals in Mumbai"
   - "Engineers across India"
   - "Specific caste in specific cities"

5. **Expand Your Criteria**
   - Don't be too strict on every filter
   - Flexibility increases chances
   - Don't compromise core values
   - But be open to surprises

6. **Review Rejected Searches**
   - Search for profiles you previously rejected
   - People grow and change
   - Preferences might change too
   - Worth revisiting

- Save your current filter combination
- Search name: "Engineering professionals in tech cities"

- Get notified of new matches for this search

### Common Search Mistakes to Avoid

| Mistake | Why it's a problem | Solution |
|---------|-------------------|----------|
| Age range: 25-26 only | Too restrictive | Expand to 24-28 |
| Only 1 city | Too narrow | Add 3-5 cities |
| All filters filled strictly | Very few results | Leave some blank |
| Income filter too high | Eliminates good people | Focus on character not money |
| Never updating search | Missing new great people | Run weekly |
| Only sorting by match % | Might miss great connections | Also browse other results |
| Too picky on appearance | Real connections matter more | Look beyond photos |
